# CardioMR - App del paciente cardiológico

## Cómo probarla en tu computadora

1. Instalá Node.js desde https://nodejs.org
2. Abrí la carpeta del proyecto en Visual Studio Code
3. Abrí una terminal y ejecutá:

npm install
npm run dev

4. Abrí el link que aparece, normalmente:
http://localhost:5173

## Cómo publicarla fácil

La opción más simple es Vercel:
1. Subir esta carpeta a GitHub
2. Entrar a vercel.com
3. Importar el repositorio
4. Deploy

Después te da un link público para compartir con pacientes.
